# 见到你很高兴

> 翻越防火长城，你可以到达世界上的每一个角落
>
> Across the Great Firewall, you can reach every corner in the world.

> 很久很久以前有一面很高的墙，墙里有两个男孩和一个女孩，他们对外面的世界充满了无限的好奇。
>
> 而现在也有一面很高的墙在我们面前，此时此刻墙里有我们。这一次，墙外又有着什么？
